Features implemented in the prototype:
	Garden:
		1. Drag the plants;
	Task Management:
		1. Show all tasks;
		2. Look at a specific task;
		3. Add/Edit/Delete a new task;
		4. Check off a task
		5. Sync a task

		2. Connect task management with plants growing;

	Task Management:
